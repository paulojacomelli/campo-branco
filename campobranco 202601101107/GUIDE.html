Campo Branco: Guia de Instalação

Siga este guia para obter as chaves necessárias para o instalador.

Passo 1

Criar Projeto no Firebase

1. Acesse console.firebase.google.com e faça login.

2. Clique em "Adicionar projeto".

3. Dê um nome (ex: minha-congregação) e siga os passos (pode desativar o Google Analytics).

Passo 2

Criar o Web App

1. Na tela inicial do projeto, clique no ícone </> (Web).

2. Dê um apelido para o app (ex: Site) e marque a opção "Firebase Hosting".

3. Clique em Registrar App, Avançar, Avançar, ir para o Console.

Passo 3

Ativar o Banco de Dados

1. No menu lateral, clique em Criação > Firestore Database.

2. Clique em Criar banco de dados.

3. Escolha o local (recomendado: sao-paulo ou us-central).

4. Escolha Iniciar no modo de produção.

Passo 4

Ativar Autenticação

1. No menu lateral, clique em Criação > Authentication.

2. Clique em Vamos começar.

3. Escolha Google > Ativar > Escolha seu email de suporte > Salvar.

Passo Final

Pegar as Chaves

1. Clique na Engrenagem (⚙️) ao lado de "Visão geral do projeto" > Configurações do projeto.

2. Role até o final da página até ver "Seus aplicativos".

3. Selecione a opção Config (não npm).

4. Você verá um código com várias chaves. Copie os valores (sem as aspas) quando o instalador pedir:

apiKey
O código longo que começa com "AIza..."
authDomain
Geralmente "seu-projeto.firebaseapp.com"
projectId
O ID do seu projeto
storageBucket
Link do storage
messagingSenderId
Número longo
appId
Código começando com 1:...
Pronto! Agora volte para a janela preta do instalador e cole essas informações quando solicitado.
